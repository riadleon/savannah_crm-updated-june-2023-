Salesforce Integration

The Salesforce integration provides a bi-directional link between your Salesforce org and Savannah CRM accounts, allowing you to share data about people and companies.

This integration will only share data about people if they are already in both systems, it will not add community contacts into Salesforce, or sales contacts into Savannah.

Step 1: Install Salesforce Package

The Savannah CRM Integration package for Salesforce provides Lightning Web Components that can be added to your Salesforce Contact and Account pages to provide extra information from Savannah. These components will need to be added to your Salesforce pages by an administrator.

Install

Step 2: Connect Savannah CRM to Salesforce

Once the package is installed to your Salesforce org you can connect your Savannah CRM account to it using the standard app authorization process.

Connect

Step 3: Connect Salesforce to Savannah CRM

Once you have connected Savannah to your Salesforce org, you will be given an API Token. This token must set as the password on the Savannah CRM API Named Credential to allow Salesforce to read data from the Savannah CRM API.
